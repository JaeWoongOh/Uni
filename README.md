#Uni FrameWork - Dynamic Async Viewer

###Add it in your root build.gradle at the end of repositories:
```
allprojects {
  repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```

<br>  
###Add the dependency
```
dependencies {
  compile 'com.github.JaeWoongOh:UniBoot:v0.5.3'
}
```
